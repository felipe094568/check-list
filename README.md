# 📋 Checklist App

Um aplicativo de lista de tarefas simples, elegante e totalmente responsivo para a organização do dia a dia.

## 🚀 Funcionalidades
- **Adicionar Tarefas:** Crie tarefas com títulos e horários personalizados.
- **Marcar como Concluída:** Risco visual nas tarefas finalizadas.
- **Editar e Excluir:** Gerencie sua lista facilmente.
- **Persistência de Dados:** Suas tarefas ficam salvas no navegador (`LocalStorage`).
- **Notificações:** Alertas agendados para avisar sobre os seus compromissos.
- **Design Responsivo:** Adaptado para celulares e computadores.

## 🛠️ Tecnologias Utilizadas
- HTML5
- CSS3 (Flexbox & Media Queries)
- JavaScript (Vanilla) & Web APIs
